My Projects

STM Gotham City Academic Management System

A web-based examination and student registration management system developed using PHP, MySQL, and TailwindCSS. It allows administrators to create, view, edit, and delete exam questions categorized by department, while also managing student registration data in a secure and user-friendly interface. The system features a modern dashboard with real-time statistics of registered students, including verification status, popular programs, and registration trends through interactive charts. Administrators can verify or delete student records via asynchronous requests, ensuring smooth user experience. The application implements security best practices such as CSRF protection, input validation, and session-based authentication. The UI is designed with a dark theme, responsive layout, and interactive elements like confirmation modals and sortable data tables.

PHP MySQL Tailwind CSS JavaScript SweetAlert2 Web Security

Network Dashboard (NetMon)

A modern network monitoring dashboard built with Python, Flask, and Chart.js. NetMon allows users to monitor real-time network speed, run internet speed tests, view public IP geolocation on an interactive map, and track historical network data with filtering and export features. The system uses SQLite for data storage and SQLAlchemy for ORM, combined with responsive HTML, CSS, and JavaScript for a clean and intuitive user interface. A powerful Bash-based toolkit for network resource fetching, monitoring, and logging, optimized for WSL Kali Linux. All activities are logged with timestamps for auditing and troubleshooting.
