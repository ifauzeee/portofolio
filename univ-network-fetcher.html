Universal Network Resource Fetcher

About the Project

A powerful Bash script for network resource fetching, monitoring, and logging, optimized for WSL Kali Linux. This interactive TUI tool combines multiple networking utilities into one comprehensive solution.

Key Features

  • File downloading with URL support
  • Public IP address checking
  • Network interface monitoring
  • Latency testing capabilities
  • Port scanning functionality
  • Google Drive direct link generation
  • Traceroute implementation
  • YouTube video downloading
